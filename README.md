# Portfolio Website

A modern personal portfolio website built using React and TailwindCSS.  
It showcases my skills, projects, and experience in a responsive design.

## 🚀 Features
- Fully responsive design (mobile, tablet, desktop)
- Modern UI with smooth layout
- About section highlighting skills and experience
- Projects showcase section
- Contact section

## 🛠️ Built With
- ReactJS
- TailwindCSS

## ⚠️ Note
This project was originally based on the following template:  
https://github.com/machadop1407/beautiful-react-tailwind-portfolio.git  

I have customized and modified it with my own changes, improvements, and personal content to fit my portfolio.
